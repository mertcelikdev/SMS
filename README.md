# Student Management System

Welcome to the **Student Management System**! This is a simple yet powerful system designed to manage both student and teacher records. It allows students and teachers to log in and interact with the system based on their roles. Students can easily select courses, while teachers can manage their assigned courses and student registrations. 

## Planned Features

- **Student and Teacher Login**: Secure login for both students and teachers to access their personalized dashboard and functionalities.
  
- **Student Course Selection**: Students can browse through available courses and select the ones they wish to enroll in.

- **Registration Management**: Teachers and administrators can efficiently manage student course registrations, ensuring everything stays organized and up-to-date.

- **Student Information**: Students can view their personal details, including enrolled courses, registration status, and more.

- **Course Information**: Both students and teachers can access detailed information about available courses, including course name, description, and teacher assignment.

- **Pagination**: To enhance navigation and improve user experience, lists of students, courses, and registrations are paginated.

- **Sorting**: Tables throughout the system can be sorted by key fields such as student names, course names, and registration statuses.
